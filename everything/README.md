# Everything Command-Line Interface Directory

This directory should contain `es.exe` from Everything.

## Download

Download Everything command-line tools from:
https://www.voidtools.com/downloads/

## Installation

1. Download the command-line interface (64-bit or 32-bit)
2. Extract `es.exe`
3. Place `es.exe` in this directory

## Usage

The Larj application will automatically use `es.exe` for file searching.

## Note

`es.exe` is not included in the repository due to licensing.
You must download it separately from the official Everything website.
