## 💡 设计亮点

### 1. 事件驱动架构
使用 PyQt5 信号-槽机制实现松耦合的模块通信

### 2. 异步搜索
使用 QThread 避免阻塞 UI，保持界面流畅

### 3. 智能缓存
搜索结果缓存 60 秒，减少重复搜索

### 4. 防抖优化
输入防抖 300ms，避免频繁触发搜索

### 5. 模块化设计
每个模块职责单一，易于维护和扩展

## 📝 使用说明

### 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/LISANG3/Larj.git
cd Larj

# 2. 安装依赖
pip install -r requirements.txt

# 3. 下载 Everything es.exe
# 访问 https://www.voidtools.com/downloads/
# 将 es.exe 放入 everything/ 目录

# 4. 启动应用
python main.py
# 或使用启动脚本
launch.bat
```

### 基本操作

1. **触发面板**: 按鼠标侧键（XButton1）
2. **搜索文件**: 在搜索框输入关键词
3. **启动应用**: 点击应用图标
4. **隐藏面板**: 按 ESC 键或点击其他区域
