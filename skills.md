# Skills 使用指南

## 任务目标
- 本 Skill 用于：指导用户安装、配置和使用各类技能包(Skills)
- 能力包含：技能包管理、领域技能推荐、最佳实践指导
- 触发条件："有哪些可用的skill"、"skill使用方法"、“如何做”等问题

## 前置准备
- 确保已安装 Node.js 环境（版本 >= 14.0）
- 确保已配置好 npm 或 npx 命令
- 检查网络连接是否正常（需要访问技能包仓库）

## 核心概念

### 什么是 Skills？
Skills 是一种可扩展的能力包系统，通过安装不同的技能包，可以为 AI 助手赋予特定领域的专业能力。每个技能包都包含经过优化的提示词、工作流程和最佳实践。

### Skills 的优势
- **模块化**：按需安装，按需加载
- **专业性**：每个技能包都针对特定领域优化
- **可扩展**：支持社区贡献和自定义开发
- **版本管理**：支持版本控制和平滑升级

## 安装方法

### 基本安装命令
所有技能包都通过 `npx skills add` 命令安装：

```bash
npx skills add <skill-package-name>
```

### 安装验证
安装完成后，可通过以下方式验证：
- 查看安装日志确认无错误信息
- 在对话中提及相关领域话题，观察AI是否自动加载技能

## 推荐技能包

详见 [技能包目录](references/skills-catalog.md)，包含以下领域的技能包：

1. **UI/UX 设计**：界面设计、交互优化、视觉效果提升
2. **插件开发**：插件体系架构、插件开发流程、插件市场运营
3. **系统优化**：性能调优、稳定性增强、响应速度优化
4. **桌面开发**：原生桌面UI开发、Windows平台适配
5. **更多技能**：持续更新的社区技能包

## 操作步骤

### 步骤1：识别需求场景
根据当前任务类型，判断需要哪个领域的技能包支持：
- 界面设计任务 → UI/UX 设计相关技能
- 开发插件功能 → 插件开发相关技能
- 性能问题排查 → 系统优化相关技能
- 桌面应用开发 → 原生UI开发相关技能

### 步骤2：安装对应技能包
使用 `npx skills add` 命令安装所需技能包。详见 [技能包目录](references/skills-catalog.md) 中的具体安装命令。

### 步骤3：验证技能加载
在对话中提出相关领域的问题，确认AI已加载对应技能：
- AI会自动识别并加载相关技能包
- 回答中会体现专业技能和最佳实践

### 步骤4：遵循技能指导
根据加载的技能包内容，按照其提供的工作流程和最佳实践执行任务。

## 注意事项

### 安装建议
- **按需安装**：只安装当前任务需要的技能包，避免过度安装
- **版本选择**：优先选择稳定版本，生产环境谨慎使用测试版
- **网络环境**：确保网络连接稳定，安装失败时可重试

### 使用建议
- **主动提及**：在对话中明确提及所需领域，帮助AI快速定位技能
- **组合使用**：复杂任务可能需要多个技能包协同工作
- **反馈问题**：遇到技能包问题，及时向社区反馈

### 常见问题

**Q: 安装失败怎么办？**
A: 检查网络连接，确认包名正确，尝试使用稳定版本

**Q: 如何查看已安装的技能包？**
A: 查看项目配置文件或询问AI当前可用的技能

**Q: 技能包会自动加载吗？**
A: AI会根据对话内容自动识别并加载相关技能，无需手动激活

**Q: 如何卸载技能包？**
A: 使用 `npx skills remove <package-name>` 命令

## 使用示例

### 示例1：UI设计优化
```bash
# 安装UI/UX设计技能包
npx skills add nextlevelbuilder/ui-ux-pro-max-skill

# 在对话中提出需求
"请帮我优化这个登录页面的用户体验"
# AI将自动加载UI/UX技能并提供专业建议
```

### 示例2：插件开发指导
```bash
# 安装插件开发技能包
npx skills add anthropics/skills

# 在对话中提出需求
"我想开发一个VS Code插件，应该从哪里开始？"
# AI将提供插件开发的完整流程和最佳实践
```

### 示例3：性能优化
```bash
# 安装系统优化技能包
npx skills add obra/superpowers

# 在对话中提出需求
"应用响应速度慢，如何进行性能优化？"
# AI将使用系统优化技能提供诊断和优化方案
```
